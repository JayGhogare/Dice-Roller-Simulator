🎲 Dice Roller Simulator

📌 About the Project

The Dice Roller Simulator is a beginner-friendly Python project that simulates rolling a real dice using random number generation.

The program waits for the user to press Enter and then generates a random number between 1 and 6, just like rolling a physical dice.

This project helped me understand Python modules, random number generation, user interaction, and basic program flow.

🎯 How It Works

1️⃣ The program asks the user to press Enter.

2️⃣ The user presses Enter to roll the dice.

3️⃣ Python generates a random number between 1 and 6.

4️⃣ The result is displayed on the screen.
